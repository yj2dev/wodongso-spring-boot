## 대학교 동아리 소개 플랫폼
- 사용자의 목소리에 어울리는 장르를 추천해주고 작사 작곡 결과를 보여주는 서비스
## 개발 기간
- 22.11 ~ 12 2개월
## 개발 인원
- 2명: 백앤드(본인), 프론트(1명)
## 개발 내용
- 각 학교에서 생성한 동아리만 보이게 처리
- 학교내에서 관리자 · 개설자 별 권한 부여
